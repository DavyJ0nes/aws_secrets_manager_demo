# AWS Secret Manager Demo

## Description

Basic demo of how to use AWS Secret Manager with boto3.

## Usage

`python secrets_demo.py`

```shell
$ python secrets_demo.py
--------------------
setting secret
secret name: test1
secret value: pnJL0583MweP0cG59dsWYk2fzcY4Hw6o
--------------------
retrieving secret
retrieved secret: pnJL0583MweP0cG59dsWYk2fzcY4Hw6o
```
